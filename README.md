# 📖 Overview

The **Loan Management System** is a desktop application designed to simplify loan related operations for businesses and organizations. It enables users to manage borrowers, process loans, monitor repayments, and organize financial records efficiently.

The application follows the **MVVM (Model View ViewModel)** architecture, making the code clean, maintainable, and scalable.

# 🏗 Project Architecture

The application follows the **MVVM Architecture**.

```text
Presentation Layer
        │
        ▼
 View (XAML)
        │
        ▼
 ViewModels
        │
        ▼
 Business Logic
        │
        ▼
 Models
        │
        ▼
 SQL Server Database
```

# 🚀 Installation

### 1 Clone Repository

```bash
git clone https://github.com/ishfaqkhan-dev/Loan_Management_System_Desktop_Application.git
```

---

### 2 Open Project

Open the project using **Visual Studio 2022**.

---

### 3 Configure Database

Create or restore the SQL Server database and update the connection string.

---

### 4 Build Project

Restore NuGet packages and build the solution.

---

### 5 Run Application

Press **F5** to launch the application.
